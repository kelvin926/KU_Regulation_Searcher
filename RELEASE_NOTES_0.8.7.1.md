# KU Regulation Searcher 0.8.7.1

## 변경 사항

- 로컬 규정 DB 기반 평가를 자연어 복합 질문 10,000개로 확장했습니다.
- 규정명을 문장 중간에서 직접 언급하는 질문도 해당 규정 조항을 우선 후보로 수집하도록 보강했습니다.
- `GKS융합전공 운영 지침 제4조...`처럼 규정명과 조항 번호를 함께 묻는 질문에서 같은 조 번호의 다른 규정이 근거로 선택되는 문제를 수정했습니다.
- 직접 지정한 규정은 `학부`, `조교`, `직원` 같은 범위 추정 페널티 때문에 AI 근거에서 빠지지 않도록 보정했습니다.
- `회의의 소집과 의결`처럼 일반적인 소집 표현이 군입대 휴학 질의로 오인되지 않도록 키워드 확장을 조정했습니다.

## 검증

- `npm test`: 통과, 11개 테스트 파일 / 53개 테스트
- `node scripts/evaluate-local-search.cjs --count 10000 --style complex --minPassRate 0.999`: 통과, 10,000/10,000
- `npm run dist:win`: Windows NSIS 설치본 빌드 검증
