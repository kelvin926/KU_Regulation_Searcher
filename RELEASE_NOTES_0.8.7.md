# KU Regulation Searcher 0.8.7

## 변경 사항

- 로컬에 저장된 규정 DB에서 5,000개 예시 질문을 생성해 검색 근거 선택을 검증하는 평가 스크립트를 추가했습니다.
- 직접 언급된 규정명을 공백과 일부 문장부호를 무시해 먼저 후보로 넣도록 검색 후보 수집을 보강했습니다.
- `고려대학교 학칙 제1조`, `구 매 규 정 제2조`, `대학원학칙 ... 시행세칙`처럼 규정명과 조항 번호를 함께 묻는 질문의 근거 검색 정확도를 개선했습니다.
- 직접 언급된 규정명/조항 제목이 있으면 `대상`, `요건`, `지급 기준`, `금액` 같은 짧고 일반적인 단어만으로도 AI 근거에서 빠지지 않도록 재정렬 기준을 보정했습니다.
- `경제학과 신임교원...`처럼 학과명이 포함된 교원 규정을 학부생 질문으로 오인하지 않도록 범위 판별 순서를 수정했습니다.

## 검증

- `npm test`: 통과, 11개 테스트 파일 / 50개 테스트
- `npm run eval:local-search -- --count 5000`: 통과, 5,000/5,000
- `npm run dist:win`: Windows NSIS 설치본 빌드 검증
