# KU Regulation Searcher 0.8.8.1

## 변경 사항

- 고려대 규정집에서 자주 쓰이는 행정 표현에 맞춰 동의어 확장을 보강했습니다.
- `학비`를 `등록금`/`수업료`/`납입금`으로, `강의시수`/`책임시수`를 `책임수업시간`으로 확장합니다.
- `강의실을 빌리다`처럼 시설·공간·장비 맥락이 있는 질문은 `대관`/`대여`/`시설사용`/`사용신청`까지 함께 찾습니다.
- `교환학생`/`방문학생`/`SEP`/`VSP`, `지도교수 변경`, `졸업유예`, `임용`/`채용` 등 규정집 표현과 사용자 표현 사이의 연결을 추가했습니다.
- `우주선을 빌릴 수 있나요?`처럼 대상 자체가 규정집에 없는 질문은 단순히 `빌리다` 때문에 대여 규정으로 과도하게 확장되지 않도록 맥락 조건을 두었습니다.

## 검증

- `npm test`: 통과, 11개 테스트 파일 / 57개 테스트
- `npm run eval:local-search:complex`: 통과, 10,000/10,000
- `npm run dist:win`: Windows NSIS 설치본 빌드 검증
