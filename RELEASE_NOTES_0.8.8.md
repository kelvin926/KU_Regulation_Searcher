# KU Regulation Searcher 0.8.8

## 변경 사항

- 복합 자연어 질문을 하나의 검색식으로만 처리하지 않고, 하위 검색 의도로 분해해 후보 근거를 넓게 수집하도록 검색 파이프라인을 개편했습니다.
- `군휴학을 냈다가 일반휴학으로 전환`, `신임 교수의 영강 의무`, `세종/서울 총학생회 차이`처럼 여러 개념이 섞인 질문에서 관련 규정이 AI 근거로 들어가도록 보정했습니다.
- `영강`을 `영어강의`/`외국어강의`로, `교수`를 `교원`으로 확장하는 등 고려대 규정집 표현에 맞춘 동의어 확장을 추가했습니다.
- 복합 질문에서는 특정 단어 하나가 빠졌다는 이유만으로 핵심 조항을 낮은 관련도로 밀어내지 않도록 재정렬 기준을 조정했습니다.
- 비교형 질문과 외국어강의 질문에서는 같은 규정 조항만 과도하게 몰리지 않도록 근거 다양성을 높였습니다.
- 로컬 검색 평가 생성기가 다른 조문번호를 본문 힌트로 뽑아 충돌 질문을 만들지 않도록 보정했습니다.

  - `신임 교수의 영강 의무는 어떻게 되지?`
