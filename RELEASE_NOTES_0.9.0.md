# KU Regulation Searcher 0.9.0

## 개선 사항

- 복합 자연어 질문을 범위, 주제, 절차, 기간, 자격, 비교, 전환 의도로 나누어 추가 검색식을 만들고 재정렬하도록 검색 파이프라인을 보강했습니다.
- 직접 규정명이 들어간 질문은 넓은 라우팅을 생략해 기존 속도를 유지합니다.
- 검색 결과가 약하거나 넓을 때 화면에 검색 분석과 재검색 예시를 표시합니다.
- 긴 조항 본문은 AI 프롬프트에서 앞뒤 핵심 구간을 보존하고 중간을 축약해 토큰 낭비를 줄입니다.
- 대학원생 자퇴, 지도교수 변경, 군휴학 전환, 총학생회 비교, 신임교원 영강 질문의 근거 선택을 보정했습니다.
- `지도교수`라는 표현을 일반 교원 질문으로 오해하지 않도록 범위 판정을 고쳤습니다.
